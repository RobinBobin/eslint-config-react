Install the following packages:

    pnpm i --save-dev @eslint/js@^9.38.0 @robinbobin/eslint-config @robinbobin/eslint-config-react @stylistic/eslint-plugin@^5.5.0 @typescript-eslint/parser@^8.46.2 eslint@^9.38.0 eslint-import-resolver-typescript@^4.4.4 eslint-plugin-import-x@^4.16.1 eslint-plugin-promise@^7.2.1 eslint-plugin-react-hooks@^7.0.1 eslint-plugin-simple-import-sort@^12.1.1 typescript-eslint@^8.46.2

Refer to [this section](https://github.com/RobinBobin/eslint-config/blob/main/README.md#config_eslint.config.js).
